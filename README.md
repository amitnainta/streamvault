# StreamVault

> Your media. Your rules. Zero paywalls.

StreamVault is an open-source home media server built from scratch with a modern stack — designed to be the media server we wish existed: Jellyfin's freedom, Plex's polish, and better performance at scale.

## Philosophy

- **Free forever** — no paywalls, no tiers, no surprise fees
- **Open source** — MIT licensed, community-first
- **Privacy first** — your data stays on your hardware
- **API-first** — every feature accessible via REST API, enabling third-party clients
- **Modern stack** — Go backend, React frontend, PostgreSQL, FFmpeg
